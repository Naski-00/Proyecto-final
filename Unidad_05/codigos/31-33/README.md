# 🔂 Prácticas 31 a 33: Estructuras Repetitivas Anidadas

Este subdirectorio aloja las soluciones lógicas avanzadas desarrolladas en equipos de 3 integrantes. El objetivo central es dominar el anidamiento de bucles (colocar un ciclo interno que se ejecute por completo en cada iteración individual de un ciclo externo) utilizando las tres variantes sintácticas de control de flujo.

## 💻 Catálogo de Códigos de Fuente

| Archivo | Descripción / Objetivo del Programa |
| :--- | :--- |
| [💻 Practica31_tablas_de_multiplicar.c](./Practica31_tablas_de_multiplicar.c) | Automatización e impresión estructurada de todas las tablas de multiplicar de forma consecutiva (desde la tabla del 1 hasta la del 10) usando estructuras **`for` anidadas**. |
| [💻 Practica32_tablas_de_multiplicar_while.c](./Practica32_tablas_de_multiplicar_while.c) | Resolución del sistema completo de matrices de multiplicación del 1 al 10 implementando exclusivamente lógica de bloques **`while` anidados**. |
| [💻 Practica33_tablas_de_multiplicar_do_while.c](./Practica33_tablas_de_multiplicar_do_while.c) | Procesamiento automatizado de las tablas aritméticas generales del 1 al 10 estructurado mediante bucles de control secuencial **`do-while` anidados**. |
